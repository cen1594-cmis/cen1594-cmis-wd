Lists and Links Assignment

In this assignment, we learned how to make and use the two different types of lists, ordered and unordered. We also learn how to use hyperlinks as well.

  • Create a new folder called "listsandlinks" in your repository folder.
  • Inside listsandlinks create a file called "index.html" and a file called "main.css"
  • In index.html create an ordered list of step by step instructions to create a new webpage on the production server.
  • Also create an unordered list of 5 or more links to websites with useful webdesign related information.
  • In the nav section, create an unordered list of links with a link back to our main index page.
  • In your main index.html file in public_html, add an unordered list of links to the nav section with a link to "listsandlinks"
  • Add an article with a heading and a paragraph description of this assignment. Also add an unordered list that contains this list of requirements.
  • Turn in your work.

Tables Assignment

In this assignment, we learned how to make tables using the elemnts of thead, tbody, and tfoot. We also learned the difference between th and td.

  • Create a new folder in your repository called "tables"
  • In the "tables" folder, create a new file called from your template "index.html"
  • Be sure to use thead, tbody, and tfoot elements in your tables.
  • Add html code to recreate the webpage below.
    • second table will require the use of the colspan attribute.
    • third table will require the use of the rowspan attribute.
  • In your repository index.html file, add an article with a heading, paragraph description and this list of requirements. Also add a link to the tables/index.html file.
  • Add a link from lists and links to this assignment as well.
  • Validate your code.
  • Turn it in.

Box Assignment

In this assignment, we learned about the various properties of the box model along with how to use the selector combinator tools to style the webpages in css.

  • Create a new folder in your repository called "css".
  • In your "css" folder, create a new file called "box.css".
  • Link the box.css stylesheet to all of your previous assignments (HTML Basics, LIsts and LInks, Tables).
  • Style the box model properties for elements in both previous assignments.
    • border
    • margin
    • padding
    • width
    • height
  • Use at least 3 of the 4 selector combinators.
    • space
    • >
    • ~
    • +
  • Style additional elements and properties as needed.
  • Save all your work.
  • Turn in your work.
