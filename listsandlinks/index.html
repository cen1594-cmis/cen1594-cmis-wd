How to make a website:

  1. Clone your repository.
  2. Open up your folder.
  3. Create an HTML file using gedit insertfilename.
  4. Copy the contents of your template.html file into the new HTML file and add your own contents.
  5. use "python -m SimpleHTTPServer" to check your code on local host
  6. Add, commit, and push your file to git
  7. Use "sftp username@cmis-hsteacher" to publish your work (make sure to open up the sftp in the folder that you created the new html file in).
  8. After typing in your password, proceed to go into the public_html folder.
  9. "Put" your new html files in that folder.
  10. Go to "sftp username@cmis-hsteacher/filename" to see if the content uploaded is the one you want.
